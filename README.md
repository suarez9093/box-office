Box Office

## Built With

- [React](https://reactjs.org/)
- [React Router](https://reactrouter.com/web/guides/quick-start)

### Installation

1. Clone the repo

```sh
git clone https://github.com/suarez9093/box-office.git
```

2. Install NPM packages

```sh
npm install
```

3. Run the application

```sh
npm run start
```
